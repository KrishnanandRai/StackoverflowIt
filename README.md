## StackOverflowIt
python script to search error of a program in stackoverflow

### How to use- 
```
$ python3 search.py file_to_check_for_error.py
```
### Requirements
this scrip requires ``beautifulsoup4`` and ``google`` packages to be installed 
use-
```
$ pip install google
$ pip install beautifulsoup4
```
